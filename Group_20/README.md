# Group_20 Compiler Project

## Prerequisites
- Linux/macOS terminal
- `gcc`
- `make`

## Build
```bash
cd Group_20
make
```

This creates the executable: `stage1exe`

## Run
```bash
./stage1exe <input_testcase.txt> <output_parse_tree.txt>
```

Example:
```bash
./stage1exe testcase3.txt out_tree.txt
```

After running, choose options from the menu shown in terminal.

## Clean
```bash
make clean
```
